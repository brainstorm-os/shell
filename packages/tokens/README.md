# @brainstorm-os/tokens

Semantic design tokens + themes for [Brainstorm](https://getbrainstorm.online) —
the single source of truth flattened into CSS custom properties
(`--color-…` / `--space-…` / `--radius-…`) that the shell and its apps style
against.

```sh
npm install @brainstorm-os/tokens
```

```ts
import { flattenTokens } from "@brainstorm-os/tokens";
```

Part of the Brainstorm OS. MIT licensed.
