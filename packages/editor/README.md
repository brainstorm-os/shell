# @brainstorm-os/editor

The [Brainstorm](https://getbrainstorm.online) rich-text editor — a
[Lexical](https://lexical.dev)-based block editor with Yjs collaboration,
transclusion, comments, media blocks, slash/mention typeaheads, and a compact
variant for composers.

```sh
npm install @brainstorm-os/editor react yjs
```

```ts
import { /* editor surface */ } from "@brainstorm-os/editor";
import "@brainstorm-os/editor/editor.css";
import "@brainstorm-os/editor/styles.css"; // component styles
```

Depends on `@brainstorm-os/sdk`; `react` and `yjs` are peer dependencies. Part
of the Brainstorm OS. MIT licensed.
